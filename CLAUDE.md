# CLAUDE.md

## Commits

Do not include co-authoring trailers. Never add `Co-Authored-By: Claude ...` or `Generated with Claude Code` lines to commit messages or PR descriptions.

## Writing style

Never use en-dashes (`–`) or em-dashes (`—`). Use a regular hyphen, a comma, or split into separate sentences instead.

This applies to all output: code comments, commit messages, PR descriptions, file contents, and chat responses.
